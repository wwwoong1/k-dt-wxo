# IBM watsonx Orchestrate 기반 AI Agent 실습 교육

## 교육 개요

본 과정은 **AI Agent의 개념 이해부터 watsonx Orchestrate(WXO)를 활용한 실습형 Agent 구현**까지 단계적으로 학습하는 7시간 교육입니다.  
오전에는 AI Agent와 WXO의 기본 개념 및 사용법을 익히고, 오후에는 실제 업무 시나리오를 기반으로 다양한 Assistant와 Agent를 직접 구현합니다.

- **교육 대상**: AI Agent 및 watsonx Orchestrate 입문자
- **교육 형태**: 이론 + 실습
- **총 교육 시간**: 7시간
- **주요 목표**
  - AI Agent의 개념과 필요성을 이해한다.
  - watsonx Orchestrate의 기본 구조와 사용법을 익힌다.
  - 업무 시나리오 기반 Agent를 직접 구성해본다.
  - Main-Sub AI Agent구조, Assistant Builder 등 다양한 구현 방식을 경험한다.

## 세부 구성

### 1. AI Agent 소개
AI Agent의 개념을 이해하고, 기존 챗봇/LLM 기반 시스템과의 차이점을 학습합니다.  
기업 업무 환경에서 왜 Agent가 필요한지, 어떤 방식으로 활용될 수 있는지 사례 중심으로 살펴봅니다.

### 2. WXO 기본 사용법
watsonx Orchestrate의 기본 UI와 주요 기능을 살펴봅니다.  
Agent/Assistant를 구성하기 전에 알아야 할 핵심 개념과 기본 사용 흐름을 익힙니다.

### 3. HR-Agent 구성
HR 업무를 예시로 하여 간단한 Agent를 직접 구성합니다.  
정책 조회, 사용자 질의 대응 등 실무형 시나리오를 기반으로 Agent 설계 방식을 학습합니다.

### 4. Intelligent Assistant 구현 (Main-Sub)
Main Assistant와 Sub Assistant 구조를 활용하여 보다 체계적인 Assistant 아키텍처를 구성합니다.  
단일 Assistant를 넘어 역할 분리와 확장 가능한 구조를 이해하는 데 초점을 둡니다.

### 5. banking-backoffice 구현
은행 백오피스 업무를 예시로 하여 보다 복합적인 업무 지원 Agent를 구현합니다.  
실제 기업 업무 자동화 관점에서 어떻게 시나리오를 설계할 수 있는지 실습합니다.

### 6. Assistant-Builder 구현
Assistant Builder를 활용하여 보다 빠르고 직관적으로 Assistant를 구성하는 방법을 학습합니다.  
실습을 통해 설정 항목과 구성 방식의 차이를 직접 확인합니다.

### 7. 실습 및 자격증 소개
앞선 세션에서 마무리하지 못한 실습을 정리하고, 질의응답을 진행합니다.  
마지막으로 관련 자격증 및 추가 학습 방향을 소개하며 교육을 마무리합니다.
